Common

Minify

JUI library only requires the user to load a double package file.
You can use the theme file and otherwise, will provide additional tools to make the theme easily. 

<link rel="stylesheet" href="dist/jui.min.css" />
<link rel="stylesheet" href="dist/jennifer.theme.min.css" />

Access to the jui class must then be configured in the markup. 

<body class="jui">
    ...
</body>

As the script works only with jQuery 1.8 or higher, it is necessary to load the jQuery library first. 

<script src="lib/jquery.min.js"></script>
<script src="dist/jui.min.js"></script>

Debug

The script can be loaded for debugging purposes, which requires the base and core files to be loaded. 

<script src="js/base.js"></script>
<script src="js/core.js"></script>

Load the util and the ui module first, then load the uix and the chart module in order. 

// base
<script src="js/base.js"></script>
<script src="js/core.js"></script>

// util
<script src="js/util/math.js"></script>
<script src="js/util/svg.js"></script>
...
// ui
<script src="js/ui/button.js"></script>
<script src="js/ui/combo.js"></script>
...
// uix
<script src="js/uix/autocomplete.js"></script>
<script src="js/uix/tab.js"></script>
...

// chart base
<script src="js/chart/draw.js"></script>
<script src="js/chart/builder.js"></script>

// chart theme
<script src="js/chart/theme/jennifer.js"></script>
<script src="js/chart/theme/dark.js"></script>
...
// chart grid
<script src="js/chart/grid/core.js"></script>
<script src="js/chart/grid/block.js"></script>
...
// chart brush
<script src="js/chart/brush/core.js"></script>
<script src="js/chart/brush/bar.js"></script>
...
// chart widget
<script src="js/chart/widget/core.js"></script>
<script src="js/chart/widget/tooltip.js"></script>
...

Module(s) that you are not using may be excluded. However, be careful as an error may occur in any module that has a dependency on an excluded file.

NodeJS

Now you can use the chart from the server. As follows: npm to install it using the command: 

npm install jui

Build & Test

To test and merge a .less file and a .js file contained in the package, run the command below.
For reference, the Grunt module, which is a build tool, requires the installation of NodeJS. npm install grunt-cli -g
cd jui
npm install
grunt

The build & test command can be performed for each function as follows.

The grunt.js command performs merge and minify on a .js file, and grunt-test performs a component test.
In addition, the grunt-class command converts a .less file into css and performs merge and minify. grunt js
grunt css
grunt test
